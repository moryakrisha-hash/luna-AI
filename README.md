# LUNA AI Chatbot 🤖

LUNA AI Chatbot is a modern and interactive AI-powered chatbot developed using **HTML, CSS, and JavaScript** with **Groq API integration** for intelligent real-time responses. The project provides a smooth chatting experience with an attractive and responsive user interface.

## 🚀 Features

* Modern and responsive UI
* Real-time AI-generated responses
* Fast chatbot interaction
* Simple and clean design
* Built using HTML, CSS, and JavaScript
* Integrated with Groq API
* User-friendly chat interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Groq API
* AI & NLP Concepts

```

## 💡 How It Works

* User enters a message in the chatbox.
* JavaScript sends the request to the Groq API.
* AI processes the input and generates a response.
* The chatbot displays the response in real time.
